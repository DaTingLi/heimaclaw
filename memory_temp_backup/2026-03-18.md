## 19:35 - vsock 通信模块完成

### 实现内容
- VsockClient：宿主机端客户端
- VsockServer：microVM 内服务端
- VsockManager：CID 分配管理器
- VsockCommand：命令构建器

### 通信协议
- Unix socket 连接（vsock.sock）
- 长度前缀（4 字节大端）
- JSON 序列化

### 支持的命令类型
- execute：执行 shell 命令
- call_tool：调用工具
- health_check：健康检查
- get_info：查询系统信息

### Firecracker 集成
- 自动配置 vsock 设备
- guest_cid 基于端口分配
- uds_path 用于宿主机连接

### 集成脚本
- scripts/build-rootfs.sh：从头构建 rootfs
- scripts/update-rootfs.sh：更新现有 rootfs
